# Workspace Backup Automation Demo

## Overview
This project demonstrates:

- AWX Survey-based backup selection
- Conditional Ansible logic
- Automated Git commit & push
- Full and Specific workspace backup

## Backup Types

1. Full Backup
2. Specific Workspace Backup

## Storage

For demo purposes, backups are stored inside this repository under:

- backups/full/
- backups/specific/

In production, object storage like S3 or MinIO is recommended.
